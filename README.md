# Mini Wordle Game

브라우저에서 실행되는 5글자 영어 단어 맞추기 게임입니다.  
실제 영어 단어 검증(API)과 한국어 뜻 표시 기능이 포함되어 있어 **학습용 Wordle**로 활용할 수 있습니다.

---

## ✨ 주요 기능

- 5글자 영어 단어 6번 안에 맞추기
- 실제 사전 API로 단어 유효성 검사
- 정답 단어는 오른쪽 사이드바에 자동 기록
- 각 정답 단어의 **한국어 뜻 자동 표시**
- 이전에 나온 단어는 다시 나오지 않도록 중복 방지
- Restart 버튼으로 새 게임 시작
- 키보드/모바일 화면 키보드 모두 지원

yaml

## ▶ 실행 방법

1. 모든 파일을 같은 폴더에 둡니다.
2. `index.html`을 브라우저로 실행합니다.
3. 단어 입력 후 Enter로 제출, Backspace로 삭제합니다.

---

## 🧠 사용 API

- 랜덤 5글자 단어  
  https://random-word-api.herokuapp.com
- 영어 단어 검증 + 정의  
  https://dictionaryapi.dev
- 영어 → 한국어 번역  
  https://mymemory.translated.net

## 📚 학습 기능

  **이전 정답 + 한국어 뜻**을 스크롤로 확인할 수 있습니다.
- 영어 단어 학습 목적에 적합합니다.
